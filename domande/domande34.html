public class ProvaCiclo {
	public static void main (String[] args) {
		int i=Integer.parseInt (JOptionPane.showInputDialog ("Immetti un intero positivo"));

			System.out.println (i);
			i--;
		}
	} 
}

  • C La classe ProvaCiclo causa un errore a tempo di compilazione, perché non è possibile stabilire prima dell'esecuzione quante iterazioni verranno eseguite all'interno del metodo main.
  • D Il metodo main contiene un esempio di ciclo definito.
